Detailed Description

Constant scalar function of vector variable. Function is evaluated according to f(x) = c where x is vector of parameters, and c is the constant scalar term (function value at x=0).

When quadratic function is created or its parameters set, the Matrix parameter of the function is interpreted as Hessian, i.e. twice the matrix of quadratic coefficients.
