This class is typically used to convert scalar functions without affine transformation of coordinates (interface IScalarFunctionUntransformed) to those that support affine transformation of coordinates. Detailed Description

Scalar function that is defined as some reference function evaluated at inverse affine-trasformed parameters. This class is typically used to convert scalar functions without affine transformation of coordinates (interface IScalarFunctionUntransformed) to those that support affine transformation of coordinates.
